# Delivery App Using Flask MySql Elasticsearch Nginx uWSGI

## Requirements

* [Docker]
* [Docker-compose]


### Run

Clone the repo

    $ git clone git@github.com:maradwan/Delivery.git
    $ docker-compose up -d
